    Why you should not download the database

    • To stay up to date.
    • To have access to more informations and the community.
    • Because a free JSON API is available since PSNDLv4.
    • To write a new software. You should use the free JSON API instead.

    Why you should download the database

    • To use old softwares like PSNDLv1 or PSNStuffX or any compatible.
    • Because you want to read the database offline.
    • Because you're afraid PSNDL is going to die tomorow. (It won't! :P)
